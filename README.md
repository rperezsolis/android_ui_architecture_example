# Application Architecture - The UI Layer - Guess It!
This is the toy app for lesson 5 of the [Android App Development in Kotlin course on Udacity](https://www.udacity.com/course/developing-android-apps-with-kotlin--ud9012).

The app implements the Observer pattern making use of [lifecycle-aware components](https://developer.android.com/topic/libraries/architecture/lifecycle) such as LiveData and ViewModel.

The app also has some examples that shows how to make use of the [Data Binding library](https://developer.android.com/topic/libraries/data-binding) to write cleaner and declarative code, calling the data holded in the viewModels directly from the xml layout files, keeping the UI controllers as lean as possible.
